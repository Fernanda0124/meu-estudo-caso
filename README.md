# meu-estudo-caso

## Descrição do Projeto 

Este é um estudo de caso que consiste em uma aplicação de listagem e detalhes de alunos, dividida em duas plataformas: **Web** (React/Vite) e **Mobile** (React Native/Expo). Ambas consomem a mesma API REST de alunos para gerenciamento de dados, e a versão web inclui testes automatizados com Vitest.

##  Tecnologias Utilizadas

| **Web** | React, Vite, React Router DOM, Axios, React Bootstrap, **Vitest, React Testing Library** |
| **Mobile** | React Native, Expo, React Navigation, Axios |

##  API de Alunos

A aplicação consome a API REST documentada via Swagger:
[https://proweb.leoproti.com.br/swagger-ui/index.html](https://proweb.leoproti.com.br/swagger-ui/index.html) 

## Instruções de Instalação e Execução 

###  Pré-requisitos
* Node.js e npm (ou yarn) instalados.
* Para Mobile: Expo CLI instalado (`npm install -g expo-cli`).

###  Funcionalidades
* Tela inicial com botões para alunos 
* Tela de detalhes do aluno selecionado 
* Navegação entre telas 
* Consumo da mesma API REST de alunos
